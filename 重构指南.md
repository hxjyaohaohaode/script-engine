## 一、重构总览

### 1.1 当前架构 → 目标架构

```
当前: 硬编码的单一流程，Agent 直接调 API，不可复用
      agents/llm_client.py 直接调 DeepSeek/MiMo API
      场景生成流程写死在 Celery 任务里

目标: 四层分离的平台架构
      ┌─────────────────────────────────────────────┐
      │  第四层: 应用层 (Applications)                │
      │  互动影游 · 网文小说 · 短视频脚本 · ...       │
      │  每个应用 = 编排模板 + 输出格式 + 特定流程     │
      └─────────────────────────────────────────────┘
                              ↓
      ┌─────────────────────────────────────────────┐
      │  第三层: 编排层 (Pipeline)                     │
      │  流水线编排 · 任务分发 · 状态机 · 人类审核     │
      └─────────────────────────────────────────────┘
                              ↓
      ┌─────────────────────────────────────────────┐
      │  第二层: Agent 层 (Agents)                     │
      │  创作 · 审计 · 状态 · 素材 · 伏笔 · 创意 · 编排│
      │  每个Agent = 模型路由 + Skills + 记忆          │
      └─────────────────────────────────────────────┘
                              ↓
      ┌─────────────────────────────────────────────┐
      │  第一层: 基础设施层 (Infrastructure)            │
      │  模型网关 · RAG · 向量库 · 存储 · 消息总线      │
      └─────────────────────────────────────────────┘
```

### 1.2 重构策略

- **由底向上**: 先基础设施 → 再 Agent 框架 → 最后编排层
- **渐进式**: 现有代码不删，新代码在 core/ 下写，跑通后再迁移
- **每轮独立**: 每轮改动完可以独立测试，不影响现有功能

### 1.3 三轮改动概览

| 轮次 | 范围 | 主要改动 | 预计工作量 |
|------|------|---------|-----------|
| 第一轮 | 基础设施层 | 模型网关 + RAG + 存储抽象 + pgvector | 大 |
| 第二轮 | Agent 框架 | Agent 基类 + 重构 7 个 Agent + 注册表 | 中 |
| 第三轮 | 编排层 | Pipeline 模板 + 状态机 + API 适配 | 中 |

```python
模型路由器: 根据任务意图 + 成本策略自动选择模型。

用法:

from enum import Enum


class CostProfile(Enum):

    BALANCED = "balanced"    # 平衡成本和质量

        "reason": "DeepSeek Embedding，中文效果好",

# 成本策略: 不同 profile 对模型选择的偏好
COST_PROFILES = {
    CostProfile.ECONOMY: {
        "prefer_fallback": True,     # 优先使用 fallback（便宜模型）
        "enable_thinking": False,    # 禁用思考模式
        "max_tokens": 2048,          # 限制输出长度
    },
    CostProfile.BALANCED: {
        "prefer_fallback": False,
        "enable_thinking": False,
        "max_tokens": 4096,
    },
    CostProfile.QUALITY: {
        "prefer_fallback": False,
        "enable_thinking": True,     # 启用思考模式
        "max_tokens": 8192,
    },
}


class ModelRouter:
    """根据意图和成本策略选择模型"""

    def select_model(self, intent: str, 
                     cost_profile: CostProfile = CostProfile.BALANCED) -> str:
        """
        选择模型。

        Args:
            intent: 任务意图，如 "write.prose"
            cost_profile: 成本策略

        Returns:
            模型名称，如 "mimo-v2-pro"
        """
        route = ROUTING_TABLE.get(intent)
        if not route:
            raise ValueError(f"Unknown intent: {intent}")

        profile = COST_PROFILES[cost_profile]

        if profile["prefer_fallback"] and route["fallback"]:
            return route["fallback"][0]
        return route["models"][0]

    def get_fallbacks(self, intent: str) -> list[str]:
        """获取降级模型列表"""
        route = ROUTING_TABLE.get(intent, {})
        return route.get("fallback", [])

    def get_thinking_config(self, intent: str, 
                           cost_profile: CostProfile) -> dict | None:
        """获取思考模式配置"""
        profile = COST_PROFILES[cost_profile]
        if not profile["enable_thinking"]:
            return None

        route = ROUTING_TABLE.get(intent, {})
        primary = route["models"][0]

        # 只有 ds-v4-pro 支持 think_max
        if primary == "ds-v4-pro":
            return {"type": "enabled", "budget": 32768}
        return None
```

```python
"""

class ModelGateway:
    """统一模型调用网关"""

    def __init__(self, redis_client=None):
        self.router = ModelRouter()
        self.redis = redis_client
        self._clients: dict[str, httpx.AsyncClient] = {}

    async def invoke(
        self,
        intent: str,
        messages: list[dict],
        cost_profile: str = "balanced",
        max_tokens: int = None,
        temperature: float = 0.7,
        use_cache: bool = True,
    ) -> ModelResponse:
        """
        统一调用入口。

        Args:
            intent: 任务意图（如 "write.prose"）
            messages: 对话消息列表
            cost_profile: 成本策略（economy/balanced/quality）
            max_tokens: 最大输出 tokens（覆盖默认值）
            temperature: 温度
            use_cache: 是否使用缓存

        Returns:
            ModelResponse
        """
        profile = CostProfile(cost_profile)
        model = self.router.select_model(intent, profile)
        config = MODEL_CONFIG[model]

        # 检查缓存
        if use_cache and self.redis:
            cache_key = self._cache_key(model, messages)
            cached = await self.redis.get(cache_key)
            if cached:
                return ModelResponse(**json.loads(cached), cached=True)

        # 构建请求
        body = self._build_request_body(
            model, messages, max_tokens, temperature, intent, profile
        )

        # 调用（带降级）
        response = await self._call_with_fallback(model, intent, body, profile)

        # 写入缓存
        if use_cache and self.redis:
            await self.redis.setex(
                cache_key, 3600, json.dumps({
                    "content": response.content,
                    "model": response.model,
                    "input_tokens": response.input_tokens,
                    "output_tokens": response.output_tokens,
                    "cost": response.cost,
                })
            )

        return response

    async def _call_with_fallback(
        self, model: str, intent: str, body: dict, profile: CostProfile
    ) -> ModelResponse:
        """调用模型，失败时自动降级"""
        models_to_try = [model] + self.router.get_fallbacks(intent)

        last_error = None
        for m in models_to_try:
            try:
                return await self._call_model(m, body)
            except Exception as e:
                last_error = e
                continue

        raise RuntimeError(
            f"All models failed for intent '{intent}': {last_error}"
        )

    async def _call_model(self, model: str, body: dict) -> ModelResponse:
        """调用单个模型 API"""
        config = MODEL_CONFIG[model]
        api_key = os.getenv(config["api_key_env"])

        client = self._get_client(config["base_url"])

        # DeepSeek 和 MiMo 都兼容 OpenAI 格式
        resp = await client.post(
            f"{config['base_url']}/chat/completions",
            json={**body, "model": model},
            headers={"Authorization": f"Bearer {api_key}"},
            timeout=300,
        )
        resp.raise_for_status()
        data = resp.json()

        usage = data.get("usage", {})
        input_tokens = usage.get("prompt_tokens", 0)
        output_tokens = usage.get("completion_tokens", 0)

        # 计算费用
        cost = self._calculate_cost(model, input_tokens, output_tokens)

        return ModelResponse(
            content=data["choices"][0]["message"]["content"],
            model=model,
            input_tokens=input_tokens,
            output_tokens=output_tokens,
            cost=cost,
        )

    def _build_request_body(
        self, model, messages, max_tokens, temperature, intent, profile
    ) -> dict:
        """构建请求体"""
        body = {
            "messages": messages,
            "temperature": temperature,
        }

        if max_tokens:
            body["max_tokens"] = max_tokens
        else:
            body["max_tokens"] = COST_PROFILES[profile]["max_tokens"]

        # 思考模式（仅 DeepSeek V4 Pro）
        thinking = self.router.get_thinking_config(intent, profile)
        if thinking and "v4-pro" in model:
            body["thinking"] = thinking

        return body

    def _calculate_cost(self, model, input_tokens, output_tokens) -> float:
        """计算调用费用（元）"""
        config = MODEL_CONFIG[model]
        input_cost = input_tokens * config["input_price"] / 1_000_000
        output_cost = output_tokens * config["output_price"] / 1_000_000
        return round(input_cost + output_cost, 6)

    def _cache_key(self, model, messages) -> str:
        """生成缓存 key"""
        content = json.dumps({"model": model, "messages": messages}, sort_keys=True)
        return f"llm_cache:{hashlib.md5(content.encode()).hexdigest()}"

    def _get_client(self, base_url: str) -> httpx.AsyncClient:
        """获取或创建 httpx 客户端"""
        if base_url not in self._clients:
            self._clients[base_url] = httpx.AsyncClient(timeout=300)
        return self._clients[base_url]
```

```python
"""
成本管理: 记录每个项目的 token 用量和费用，支持预算告警。
"""

from datetime import datetime, timedelta
from typing import Optional


class CostManager:
    """成本管理器"""

    def __init__(self, db):
        self.db = db

    async def record_usage(
        self,
        project_id: str,
        agent_name: str,
        intent: str,
        model: str,
        input_tokens: int,
        output_tokens: int,
        cost: float,
    ):
        """记录一次调用的用量"""
        await self.db.execute(
            """
            INSERT INTO token_usage 
            (project_id, agent_name, intent, model, input_tokens, output_tokens, cost, created_at)
            VALUES ($1, $2, $3, $4, $5, $6, $7, $8)
            """,
            project_id, agent_name, intent, model,
            input_tokens, output_tokens, cost, datetime.utcnow()
        )

    async def get_project_cost(
        self, project_id: str, 
        since: Optional[datetime] = None
    ) -> dict:
        """获取项目的总费用"""
        if since is None:
            since = datetime.min

        row = await self.db.fetchrow(
            """
            SELECT 
                SUM(cost) as total_cost,
                SUM(input_tokens) as total_input_tokens,
                SUM(output_tokens) as total_output_tokens,
                COUNT(*) as total_calls
            FROM token_usage
            WHERE project_id = $1 AND created_at >= $2
            """,
            project_id, since
        )
        return dict(row) if row else {}

    async def get_cost_breakdown(self, project_id: str) -> list[dict]:
        """按 Agent 和模型分组的费用明细"""
        return await self.db.fetch(
            """
            SELECT 
                agent_name, model,
                SUM(cost) as total_cost,
                SUM(input_tokens) as input_tokens,
                SUM(output_tokens) as output_tokens,
                COUNT(*) as call_count
            FROM token_usage
            WHERE project_id = $1
            GROUP BY agent_name, model
            ORDER BY total_cost DESC
            """,
            project_id
        )

    async def check_budget(self, project_id: str, budget: float) -> dict:
        """检查是否接近预算"""
        cost_data = await self.get_project_cost(project_id)
        total = cost_data.get("total_cost", 0)

        return {
            "total_cost": total,
            "budget": budget,
            "remaining": budget - total,
            "usage_percent": (total / budget * 100) if budget > 0 else 0,
            "alert": total > budget * 0.8,  # 超过80%告警
        }
```

```python
"""
文本分块器: 将不同类型的文本按最优策略切分为 chunks。
"""

from dataclasses import dataclass
from typing import Literal


@dataclass
class Chunk:
    text: str
    metadata: dict
    chunk_type: str  # narration / dialogue / foreshadow / choice / field / generic


class TextChunker:
    """文本分块器，支持多种策略"""

    def chunk(self, text: str, content_type: str, 
              metadata: dict = None) -> list[Chunk]:
        """
        根据内容类型选择分块策略。

        Args:
            text: 原始文本
            content_type: 内容类型 (scene/character/foreshadow/world/generic)
            metadata: 附加元数据

        Returns:
            Chunk 列表
        """
        metadata = metadata or {}
        strategy = {
            "scene": self._chunk_scene,
            "character": self._chunk_character,
            "foreshadow": self._chunk_foreshadow,
            "world": self._chunk_world,
        }.get(content_type, self._chunk_generic)

        return strategy(text, metadata)

    def _chunk_scene(self, text: str, metadata: dict) -> list[Chunk]:
        """
        场景分块: 按结构拆分。
        场景包含: 描述、对白、伏笔操作、选择节点、因果链。
        """
        chunks = []

        # 简化实现: 按段落分块，每 500 字一个 chunk
        # 实际实现应解析 YAML/JSON 结构
        paragraphs = text.split("\n\n")
        current_chunk = ""
        current_type = "narration"

        for para in paragraphs:
            para = para.strip()
            if not para:
                continue

            # 判断段落类型
            if para.startswith("对白") or para.startswith("dialogue"):
                chunk_type = "dialogue"
            elif para.startswith("伏笔") or para.startswith("foreshadow"):
                chunk_type = "foreshadow"
            elif para.startswith("选择") or para.startswith("choice"):
                chunk_type = "choice"
            elif para.startswith("因果") or para.startswith("causal"):
                chunk_type = "causal"
            else:
                chunk_type = "narration"

            # 类型切换时保存当前 chunk
            if chunk_type != current_type and current_chunk:
                chunks.append(Chunk(
                    text=current_chunk.strip(),
                    metadata={**metadata, "chunk_type": current_type},
                    chunk_type=current_type,
                ))
                current_chunk = ""

            current_chunk += para + "\n\n"
            current_type = chunk_type

            # 超过 500 字强制切分
            if len(current_chunk) > 500:
                chunks.append(Chunk(
                    text=current_chunk.strip(),
                    metadata={**metadata, "chunk_type": current_type},
                    chunk_type=current_type,
                ))
                current_chunk = ""

        # 最后一个 chunk
        if current_chunk.strip():
            chunks.append(Chunk(
                text=current_chunk.strip(),
                metadata={**metadata, "chunk_type": current_type},
                chunk_type=current_type,
            ))

        return chunks

    def _chunk_character(self, text: str, metadata: dict) -> list[Chunk]:
        """角色分块: 按字段拆分（背景/动机/恐惧/语言风格等）"""
        # 按字段名分割
        fields = ["背景", "动机", "恐惧", "表面形象", "真实面目", 
                   "语言风格", "口头禅", "必然行为", "绝对不会", "需要铺垫"]
        chunks = []
        current_field = "basic"
        current_text = ""

        for line in text.split("\n"):
            matched = False
            for field in fields:
                if field in line:
                    if current_text.strip():
                        chunks.append(Chunk(
                            text=current_text.strip(),
                            metadata={**metadata, "field": current_field},
                            chunk_type="field",
                        ))
                    current_field = field
                    current_text = line + "\n"
                    matched = True
                    break
            if not matched:
                current_text += line + "\n"

        if current_text.strip():
            chunks.append(Chunk(
                text=current_text.strip(),
                metadata={**metadata, "field": current_field},
                chunk_type="field",
            ))
        return chunks

    def _chunk_foreshadow(self, text: str, metadata: dict) -> list[Chunk]:
        """伏笔分块: 按三层结构拆分（表面层/深层/真相层）"""
        layers = ["表面层", "深层", "真相层", "埋设", "强化", "揭露"]
        chunks = []
        current_layer = "overview"
        current_text = ""

        for line in text.split("\n"):
            matched = False
            for layer in layers:
                if layer in line:
                    if current_text.strip():
                        chunks.append(Chunk(
                            text=current_text.strip(),
                            metadata={**metadata, "layer": current_layer},
                            chunk_type="foreshadow",
                        ))
                    current_layer = layer
                    current_text = line + "\n"
                    matched = True
                    break
            if not matched:
                current_text += line + "\n"

        if current_text.strip():
            chunks.append(Chunk(
                text=current_text.strip(),
                metadata={**metadata, "layer": current_layer},
                chunk_type="foreshadow",
            ))
        return chunks

    def _chunk_world(self, text: str, metadata: dict) -> list[Chunk]:
        """世界观分块: 按主题拆分"""
        return self._sliding_window(text, metadata, window=500, overlap=100)

    def _chunk_generic(self, text: str, metadata: dict) -> list[Chunk]:
        """通用分块: 滑动窗口"""
        return self._sliding_window(text, metadata, window=500, overlap=100)

    def _sliding_window(self, text: str, metadata: dict, 
                        window: int = 500, overlap: int = 100) -> list[Chunk]:
        """滑动窗口分块"""
        chunks = []
        start = 0
        while start < len(text):
            end = start + window
            chunk_text = text[start:end]
            chunks.append(Chunk(
                text=chunk_text,
                metadata={**metadata, "position": start},
                chunk_type="generic",
            ))
            start = end - overlap
        return chunks
```

```python
import os
import httpx


class Embedder:
    """DeepSeek Embedding 调用"""

    def __init__(self):
        self.base_url = "https://api.deepseek.com/v1"
        self.api_key = os.getenv("DEEPSEEK_API_KEY")
        self.dimension = 1536

    async def embed(self, text: str) -> list[float]:
        """单条文本向量化"""
        async with httpx.AsyncClient(timeout=60) as client:
            resp = await client.post(
                f"{self.base_url}/embeddings",
                json={"input": text, "model": "deepseek-embedding"},
                headers={"Authorization": f"Bearer {self.api_key}"},
            )
            resp.raise_for_status()
            return resp.json()["data"][0]["embedding"]

    async def embed_batch(self, texts: list[str]) -> list[list[float]]:
        """批量向量化（DeepSeek 支持批量输入）"""
        async with httpx.AsyncClient(timeout=120) as client:
            resp = await client.post(
                f"{self.base_url}/embeddings",
                json={"input": texts, "model": "deepseek-embedding"},
                headers={"Authorization": f"Bearer {self.api_key}"},
            )
            resp.raise_for_status()
            # 按 index 排序确保顺序正确
            data = sorted(resp.json()["data"], key=lambda x: x["index"])
            return [item["embedding"] for item in data]
```

```python
"""
RAG 索引器: 在内容定稿后自动建立向量索引。
"""

from .chunker import TextChunker
from .embedder import Embedder


class RAGIndexer:
    """RAG 索引器"""

    def __init__(self, db):
        self.db = db
        self.chunker = TextChunker()
        self.embedder = Embedder()

    async def index_content(
        self,
        project_id: str,
        content_type: str,
        content_id: str,
        text: str,
        metadata: dict = None,
    ):
        """
        为一条内容建立向量索引。

        Args:
            project_id: 项目 ID
            content_type: 内容类型 (scene/character/foreshadow/world/info)
            content_id: 内容原始记录 ID
            text: 内容文本
            metadata: 附加元数据
        """
        metadata = metadata or {}

        # 1. 分块
        chunks = self.chunker.chunk(text, content_type, metadata)

        if not chunks:
            return

        # 2. 批量向量化
        texts = [c.text for c in chunks]
        embeddings = await self.embedder.embed_batch(texts)

        # 3. 删除该项目该内容的旧索引（重建）
        await self.db.execute(
            "DELETE FROM embeddings WHERE project_id = $1 AND content_id = $2",
            project_id, content_id
        )

        # 4. 写入新索引
        for chunk, embedding in zip(chunks, embeddings):
            await self.db.execute(
                """
                INSERT INTO embeddings 
                (project_id, content_type, content_id, chunk_text, embedding, metadata)
                VALUES ($1, $2, $3, $4, $5, $6)
                """,
                project_id, content_type, content_id,
                chunk.text, embedding, 
                {**chunk.metadata, "chunk_type": chunk.chunk_type}
            )

    async def reindex_project(self, project_id: str):
        """全量重建一个项目的索引"""
        # 清除旧索引
        await self.db.execute(
            "DELETE FROM embeddings WHERE project_id = $1", project_id
        )

        # 重建场景索引
        scenes = await self.db.fetch(
            "SELECT id, scene_code, narration, dialogue FROM scenes WHERE project_id = $1",
            project_id
        )
        for scene in scenes:
            text = f"场景 {scene['scene_code']}\n{scene['narration'] or ''}"
            if scene.get("dialogue"):
                import json
                dialogues = json.loads(scene["dialogue"]) if isinstance(scene["dialogue"], str) else scene["dialogue"]
                for d in dialogues:
                    text += f"\n{d.get('char', '')}: {d.get('text', '')}"
            await self.index_content(
                project_id, "scene", str(scene["id"]), text,
                {"scene_code": scene["scene_code"]}
            )

        # 重建角色索引
        characters = await self.db.fetch(
            "SELECT id, name, background, core_goal, language_style FROM characters WHERE project_id = $1",
            project_id
        )
        for char in characters:
            text = f"角色: {char['name']}\n背景: {char['background'] or ''}\n目标: {char['core_goal'] or ''}\n语言风格: {char['language_style'] or ''}"
            await self.index_content(
                project_id, "character", str(char["id"]), text,
                {"name": char["name"]}
            )

        # 重建伏笔索引
        foreshadows = await self.db.fetch(
            "SELECT id, fs_code, name, surface_layer, deep_layer, truth_layer FROM foreshadows WHERE project_id = $1",
            project_id
        )
        for fs in foreshadows:
            text = f"伏笔 {fs['fs_code']}: {fs['name']}\n表面: {fs['surface_layer'] or ''}\n深层: {fs['deep_layer'] or ''}\n真相: {fs['truth_layer'] or ''}"
            await self.index_content(
                project_id, "foreshadow", str(fs["id"]), text,
                {"fs_code": fs["fs_code"], "name": fs["name"]}
            )
```

```python
"""
RAG 检索器: 根据查询语义检索最相关的上下文片段。
"""

from dataclasses import dataclass
from .embedder import Embedder


@dataclass
class RetrievedChunk:
    """检索结果"""
    text: str
    content_type: str
    content_id: str
    score: float          # 余弦相似度 0-1
    metadata: dict


class RAGRetriever:
    """RAG 检索器"""

    def __init__(self, db):
        self.db = db
        self.embedder = Embedder()

    async def retrieve(
        self,
        project_id: str,
        query: str,
        content_types: list[str] = None,
        top_k: int = 5,
    ) -> list[RetrievedChunk]:
        """
        通用语义检索。

        Args:
            project_id: 项目 ID
            query: 查询文本
            content_types: 过滤的内容类型（None 表示全部）
            top_k: 返回数量

        Returns:
            按相似度排序的 RetrievedChunk 列表
        """
        query_embedding = await self.embedder.embed(query)

        # 构建过滤条件
        type_filter = ""
        params = [project_id, str(query_embedding), top_k]
        if content_types:
            placeholders = ", ".join(f"${i+4}" for i in range(len(content_types)))
            type_filter = f"AND content_type IN ({placeholders})"
            params.extend(content_types)

        rows = await self.db.fetch(
            f"""
            SELECT 
                content_type, content_id, chunk_text, metadata,
                1 - (embedding <=> $2) as score
            FROM embeddings
            WHERE project_id = $1 {type_filter}
            ORDER BY embedding <=> $2
            LIMIT $3
            """,
            *params
        )

        return [
            RetrievedChunk(
                text=row["chunk_text"],
                content_type=row["content_type"],
                content_id=row["content_id"],
                score=row["score"],
                metadata=row["metadata"],
            )
            for row in rows
        ]

    async def retrieve_for_scene(
        self,
        project_id: str,
        scene_id: str,
        requirements: dict,
    ) -> list[RetrievedChunk]:
        """
        为场景生成检索上下文（多路召回）。

        Args:
            project_id: 项目 ID
            scene_id: 目标场景 ID
            requirements: 场景需求（scene_type, emotion_target, foreshadow_tasks, character_ids 等）

        Returns:
            按相关度排序的 RetrievedChunk 列表
        """
        # 1. 构建查询描述
        query_parts = [f"场景 {scene_id}"]
        if requirements.get("scene_type"):
            query_parts.append(f"类型: {requirements['scene_type']}")
        if requirements.get("emotion_target"):
            query_parts.append(f"情感强度: {requirements['emotion_target']}/10")
        for fs_task in requirements.get("foreshadow_tasks", []):
            query_parts.append(f"伏笔 {fs_task.get('fs_id', '')} {fs_task.get('op', '')}")
        if requirements.get("is_wow_moment"):
            query_parts.append(f"哇塞时刻 {requirements.get('wow_type', '')}")

        query = "，".join(query_parts)
        query_embedding = await self.embedder.embed(query)

        # 2. 多路召回
        results = []

        # 路径 1: 相关场景（文风参考，top 3）
        scene_chunks = await self._search_by_type(
            project_id, query_embedding, "scene", top_k=3
        )
        results.extend(scene_chunks)

        # 路径 2: 出场角色设定（top 5）
        char_ids = requirements.get("character_ids", [])
        if char_ids:
            char_chunks = await self._search_by_ids(
                project_id, query_embedding, "character", char_ids, top_k=5
            )
            results.extend(char_chunks)
        else:
            char_chunks = await self._search_by_type(
                project_id, query_embedding, "character", top_k=5
            )
            results.extend(char_chunks)

        # 路径 3: 相关伏笔（top 3）
        fs_chunks = await self._search_by_type(
            project_id, query_embedding, "foreshadow", top_k=3
        )
        results.extend(fs_chunks)

        # 路径 4: 世界观条目（top 2）
        world_chunks = await self._search_by_type(
            project_id, query_embedding, "world", top_k=2
        )
        results.extend(world_chunks)

        # 3. 去重（同一个 content_id 只保留 score 最高的）
        seen = {}
        for chunk in results:
            key = chunk.content_id
            if key not in seen or chunk.score > seen[key].score:
                seen[key] = chunk

        # 4. 按 score 排序
        return sorted(seen.values(), key=lambda x: -x.score)

    async def _search_by_type(
        self, project_id, query_embedding, content_type, top_k
    ) -> list[RetrievedChunk]:
        """按内容类型检索"""
        rows = await self.db.fetch(
            """
            SELECT content_id, chunk_text, metadata, 
                   1 - (embedding <=> $3) as score
            FROM embeddings
            WHERE project_id = $1 AND content_type = $2
            ORDER BY embedding <=> $3
            LIMIT $4
            """,
            project_id, content_type, str(query_embedding), top_k
        )
        return [
            RetrievedChunk(
                text=row["chunk_text"],
                content_type=content_type,
                content_id=row["content_id"],
                score=row["score"],
                metadata=row["metadata"],
            )
            for row in rows
        ]

    async def _search_by_ids(
        self, project_id, query_embedding, content_type, ids, top_k
    ) -> list[RetrievedChunk]:
        """在指定 ID 范围内检索"""
        placeholders = ", ".join(f"${i+4}" for i in range(len(ids)))
        rows = await self.db.fetch(
            f"""
            SELECT content_id, chunk_text, metadata,
                   1 - (embedding <=> $3) as score
            FROM embeddings
            WHERE project_id = $1 AND content_type = $2 AND content_id::text IN ({placeholders})
            ORDER BY embedding <=> $3
            LIMIT ${4 + len(ids)}
            """,
            project_id, content_type, str(query_embedding),
            *[str(i) for i in ids], top_k
        )
        return [
            RetrievedChunk(
                text=row["chunk_text"],
                content_type=content_type,
                content_id=row["content_id"],
                score=row["score"],
                metadata=row["metadata"],
            )
            for row in rows
        ]
```

```python
                narration = $2, dialogue = $3, actions = $4,

        return await self.db.fetch(
            "SELECT * FROM chapters WHERE project_id = $1 ORDER BY chapter_number",
            project_id

        """获取章节上下文"""
        chapter = await self.db.fetchrow(
            "SELECT * FROM chapters WHERE id = $1", chapter_id
        )
        scenes = await self.db.fetch(
            "SELECT * FROM scenes WHERE chapter_id = $1 ORDER BY scene_code",
            chapter_id
        )
        return {"chapter": dict(chapter) if chapter else None, "scenes": [dict(s) for s in scenes]}

    async def get_scene_summaries(self, project_id: str, 
                                   scene_ids: list[str] = None) -> list[dict]:

        if scene_ids:
            return await self.db.fetch(
                "SELECT id, scene_code, narration FROM scenes WHERE project_id = $1 AND id = ANY($2)",
                project_id, scene_ids
            )
        return await self.db.fetch(
            "SELECT id, scene_code, narration FROM scenes WHERE project_id = $1",
            project_id
        )

    # ── 元素注册表 ──

    async def get_elements(self, project_id: str, 
                           element_type: str = None) -> list[dict]:
        """获取元素注册表"""
        if element_type:
            return await self.db.fetch(
                "SELECT * FROM elements WHERE project_id = $1 AND element_type = $2",
                project_id, element_type
            )
        return await self.db.fetch(
            "SELECT * FROM elements WHERE project_id = $1", project_id
        )

    async def register_element(self, project_id: str, element: dict):
        """注册新元素"""
        await self.db.execute(
            """
            INSERT INTO elements (project_id, element_type, element_code, name, description)
            VALUES ($1, $2, $3, $4, $5)
            ON CONFLICT (project_id, element_code) DO NOTHING
            """,
            project_id, element["type"], element["code"], 
            element["name"], element.get("description")
        )
```

```python
"""
Agent 基类: 所有 Agent 的统一接口。

每个 Agent 必须:
  1. 声明 name（唯一标识）和 description（人类可读描述）
  2. 实现 skills 字典（skill_name → Skill 实例）
  3. 实现 _validate, _build_context, _select_skill 方法
"""

from abc import ABC, abstractmethod
from dataclasses import dataclass, field
from typing import Any, Optional

from core.gateway.client import ModelGateway, ModelResponse
from core.rag.retriever import RAGRetriever
from core.storage.service import StorageService


@dataclass
class AgentTask:
    """Agent 任务"""
    task_id: str
    agent_name: str
    task_type: str              # 如 "write_scene", "audit_scene"
    project_id: str
    payload: dict               # 任务参数
    cost_profile: str = "balanced"  # economy / balanced / quality


@dataclass
class AgentResult:
    """Agent 执行结果"""
    status: str                 # completed / failed / needs_human / reject
    data: dict                  # 输出数据
    token_usage: Optional[dict] = None  # {input_tokens, output_tokens, cost}
    issues: list = field(default_factory=list)  # 问题列表（审计用）


class BaseAgent(ABC):
    """Agent 基类"""

    name: str                   # 唯一标识，如 "creator", "auditor"
    description: str            # 人类可读描述
    skills: dict[str, Any]      # {skill_name: Skill 实例}
    checkers: list = []         # 程序化检测器列表（审计 Agent 用）

    def __init__(self, gateway: ModelGateway, rag: RAGRetriever, 
                 storage: StorageService):
        self.gateway = gateway
        self.rag = rag
        self.storage = storage

    async def execute(self, task: AgentTask) -> AgentResult:
        """
        统一执行入口。
        子类一般不需要覆盖此方法，除非有特殊流程（如审计 Agent 先跑程序化检测）。
        """
        # 1. 校验输入
        self._validate(task)

        # 2. 组装上下文
        context = await self._build_context(task)

        # 3. 选择 Skill
        skill = self._select_skill(task.task_type)

        # 4. 执行 Skill
        result_data = await skill.execute(
            context=context,
            requirements=task.payload,
            gateway=self.gateway,
            cost_profile=task.cost_profile,
        )

        # 5. 后处理
        await self._post_process(task, result_data)

        return AgentResult(
            status="completed",
            data=result_data,
        )

    @abstractmethod
    def _validate(self, task: AgentTask):
        """校验任务输入，不合法时抛出异常"""
        ...

    @abstractmethod
    async def _build_context(self, task: AgentTask) -> dict:
        """组装上下文，返回 Skill 需要的所有变量"""
        ...

    @abstractmethod
    def _select_skill(self, task_type: str):
        """根据任务类型选择 Skill"""
        ...

    async def _post_process(self, task: AgentTask, result: dict):
        """后处理（子类可覆盖）"""
        pass
```

#### 3.1.2 backend/core/agent/skill.py

```python
"""
Skill 基类: Agent 内部的最小能力单元。

一个 Skill = 一段 prompt 模板 + 一个模型意图 + 一个输出解析器。
"""

from typing import Callable
from core.gateway.client import ModelGateway


class Skill:
    """Skill 基类"""

    name: str                       # Skill 名称
    intent: str                     # 模型意图（对应 ModelGateway 的 intent）
    prompt_template: str            # prompt 模板（支持 {variable} 占位符）
    output_parser: Callable         # 输出解析函数: str → dict

    async def execute(
        self,
        context: dict,
        requirements: dict,
        gateway: ModelGateway,
        cost_profile: str = "balanced",
    ) -> dict:
        """
        执行 Skill。

        Args:
            context: 上下文变量（用于填充 prompt_template）
            requirements: 任务需求参数
            gateway: 模型网关
            cost_profile: 成本策略

        Returns:
            解析后的输出 dict
        """
        # 1. 渲染 prompt
        prompt = self._render(context, requirements)

        # 2. 调用模型
        response = await gateway.invoke(
            intent=self.intent,
            messages=[{"role": "user", "content": prompt}],
            cost_profile=cost_profile,
        )

        # 3. 解析输出
        return self.output_parser(response.content)

    def _render(self, context: dict, requirements: dict) -> str:
        """渲染 prompt 模板"""
        # 合并 context 和 requirements，优先 requirements
        variables = {**context, **requirements}
        try:
            return self.prompt_template.format(**variables)
        except KeyError as e:
            # 缺少变量时用空字符串填充
            import re
            missing = re.findall(r'\{(\w+)\}', self.prompt_template)
            safe_vars = {k: variables.get(k, "") for k in missing}
            return self.prompt_template.format(**safe_vars)
```

```python
AGENT_REGISTRY: dict[str, type] = {}


def register_agent(cls):
    """装饰器: 注册 Agent 类"""
    AGENT_REGISTRY[cls.name] = cls
    return cls


def get_agent(name: str, gateway, rag, storage):
    """根据名称获取 Agent 实例"""
    cls = AGENT_REGISTRY.get(name)
    if not cls:
        raise ValueError(f"Unknown agent: {name}. Available: {list(AGENT_REGISTRY.keys())}")
    return cls(gateway, rag, storage)


def list_agents() -> list[dict]:
    """列出所有已注册的 Agent"""
    return [
        {"name": cls.name, "description": cls.description}
        for cls in AGENT_REGISTRY.values()
    ]
```

```python
@register_agent
class CreatorAgent(BaseAgent):
    name = "creator"
    description = "场景创作、对白生成、分支设计、世界观构建、角色设计、大纲撰写"
    skills = {
        "scene_writer": SCENE_WRITER_SKILL,
        "dialogue_writer": DIALOGUE_WRITER_SKILL,
        "branch_designer": BRANCH_DESIGNER_SKILL,
        "world_builder": WORLD_BUILDER_SKILL,
        "character_designer": CHARACTER_DESIGNER_SKILL,
        "outline_writer": OUTLINE_WRITER_SKILL,
    }

    def _validate(self, task: AgentTask):
        if not task.project_id:
            raise ValueError("project_id is required")
        if task.task_type not in self.skills:
            raise ValueError(f"Unknown task_type: {task.task_type}. Available: {list(self.skills.keys())}")

    async def _build_context(self, task: AgentTask) -> dict:
        """组装上下文: 基础层 + RAG 召回"""
        project_id = task.project_id
        payload = task.payload

        # 基础层: 项目设定
        layer0 = await self.storage.get_layer0(project_id)

        # RAG 召回: 与当前任务最相关的上下文片段
        rag_query = self._build_rag_query(task)
        rag_results = await self.rag.retrieve(
            project_id, rag_query, top_k=10
        )

        # 组装 context
        context = {
            "layer0": self._format_layer0(layer0),
            "rag_context": "\n---\n".join(r.text for r in rag_results),
        }

        # 根据任务类型补充特定上下文
        if task.task_type == "scene_writer":
            scene_id = payload.get("scene_id")
            if scene_id:
                prev_scenes = await self.storage.get_prev_scenes(scene_id, count=2)
                context["previous_scene"] = self._format_scenes(prev_scenes)

                scene = await self.storage.get_scene(project_id, scene_id)
                if scene:
                    context["scene_code"] = scene.get("scene_code", "")
                    context["genre"] = layer0.get("genre", {}).get("value", "")
                    context["style"] = layer0.get("style", {}).get("value", "")

            # 角色状态
            char_ids = payload.get("character_ids", [])
            chars = await self.storage.get_character_states(project_id, char_ids)
            context["character_states"] = self._format_characters(chars)

        return context

    def _select_skill(self, task_type: str) -> Skill:
        return self.skills[task_type]

    def _build_rag_query(self, task: AgentTask) -> str:
        """构建 RAG 查询"""
        parts = []
        if task.task_type == "scene_writer":
            parts.append(f"场景 {task.payload.get('scene_id', '')}")
            parts.append(f"类型 {task.payload.get('scene_type', '')}")
            parts.append(f"情感 {task.payload.get('emotion_target', 5)}/10")
        elif task.task_type == "world_builder":
            parts.append(task.payload.get("core_contradiction", ""))
        return "，".join(filter(None, parts)) or task.project_id

    def _format_layer0(self, layer0: dict) -> str:
        """格式化 Layer 0 为文本"""
        parts = []
        for key, val in layer0.items():
            parts.append(f"【{key}】\n{val.get('value', val) if isinstance(val, dict) else val}")
        return "\n\n".join(parts)

    def _format_scenes(self, scenes: list) -> str:
        """格式化场景列表为文本"""
        parts = []
        for s in scenes:
            parts.append(f"场景 {s.get('scene_code', '?')}:\n{s.get('narration', '')}")
        return "\n---\n".join(parts)

    def _format_characters(self, chars: list) -> str:
        """格式化角色信息为文本"""
        parts = []
        for c in chars:
            parts.append(f"{c.get('name', '?')} ({c.get('role_type', '?')}): 背景={c.get('background', '')}, 动机={c.get('core_goal', '')}")
        return "\n".join(parts)
```

#### 3.2.2 其他 Agent 的重构模式

**所有 Agent 遵循相同模式:**

1. 定义 Skill 实例（name + intent + prompt_template + output_parser）
2. 定义 Agent 类继承 BaseAgent
3. 用 @register_agent 装饰
4. 实现 _validate, _build_context, _select_skill

**审计 Agent 额外需要:**
- 重写 `execute` 方法，先跑 `self.checkers`（程序化检测），全通过才调 LLM
- checkers 列表包含现有的 6 个检测器

**素材 Agent 不调用 LLM:**
- 直接操作 `self.rag` 和 `self.storage`
- execute 方法直接返回检索结果，不走 gateway

#### 4.1.1 backend/core/pipeline/template.py

```python
"""
编排模板: 定义 Agent 的执行顺序、条件分支、人类审核节点。
不同应用场景 = 不同模板。
"""

from dataclasses import dataclass, field


@dataclass
class Step:
    """一个执行步骤"""
    agent: str                  # Agent 名称
    skill: str                  # Skill 名称
    input_from: list[str]       # 输入来源: ["user_input", "layer0", "layer1", "rag", "prev_step"]
    output_to: str              # 输出去向: "layer0" / "layer1" / "layer2" / "layer5" / "next_step"
    condition: str = None       # 执行条件（可选），如 "emotion_target >= 7"
    cost_profile: str = "balanced"
    timeout: int = 600


@dataclass
class Phase:
    """一个阶段"""
    name: str
    steps: list[Step]
    human_gate: bool = False    # 是否需要人类审核才能进入下一阶段
    max_parallel: int = 1       # 最大并行数
    repeat_until: str = None    # 循环条件（如 "all_scenes_done"）


@dataclass
class PipelineTemplate:
    """编排模板"""
    name: str
    description: str
    phases: list[Phase]
    scale_config: dict = field(default_factory=dict)  # 轻量/标准/重型参数覆盖
```

#### 4.1.2 backend/core/pipeline/state_machine.py

```python
"""
流程状态机: 管理一个项目的执行状态。
"""

from dataclasses import dataclass
from enum import Enum
from typing import Optional
import json


class PipelineStatus(Enum):
    NOT_STARTED = "not_started"
    RUNNING = "running"
    WAITING_HUMAN = "waiting_human"
    COMPLETED = "completed"
    FAILED = "failed"


class StepStatus(Enum):
    PENDING = "pending"
    RUNNING = "running"
    COMPLETED = "completed"
    FAILED = "failed"
    SKIPPED = "skipped"


@dataclass
class PipelineState:
    """项目流程状态"""
    project_id: str
    template_name: str
    status: PipelineStatus
    current_phase_index: int
    current_step_index: int
    completed_phases: list[int]
    completed_steps: dict[str, list[int]]  # {phase_index: [step_indices]}
    rejection_counts: dict[str, int]       # {task_key: count}


class PipelineStateMachine:
    """流程状态机"""

    def __init__(self, db):
        self.db = db

    async def init(self, project_id: str, template_name: str):
        """初始化流程状态"""
        await self.db.execute(
            """
            INSERT INTO pipeline_state 
            (project_id, template_name, status, current_phase_index, current_step_index,
             completed_phases, completed_steps, rejection_counts, created_at, updated_at)
            VALUES ($1, $2, 'not_started', 0, 0, '[]', '{}', '{}', NOW(), NOW())
            ON CONFLICT (project_id) DO NOTHING
            """,
            project_id, template_name
        )

    async def get_state(self, project_id: str) -> Optional[PipelineState]:
        """获取流程状态"""
        row = await self.db.fetchrow(
            "SELECT * FROM pipeline_state WHERE project_id = $1", project_id
        )
        if not row:
            return None
        return PipelineState(
            project_id=row["project_id"],
            template_name=row["template_name"],
            status=PipelineStatus(row["status"]),
            current_phase_index=row["current_phase_index"],
            current_step_index=row["current_step_index"],
            completed_phases=json.loads(row["completed_phases"]),
            completed_steps=json.loads(row["completed_steps"]),
            rejection_counts=json.loads(row["rejection_counts"]),
        )

    async def advance_step(self, project_id: str):
        """推进到下一步"""
        state = await self.get_state(project_id)
        if not state:
            return

        template = get_template(state.template_name)
        phase = template.phases[state.current_phase_index]

        # 标记当前步骤完成
        if state.current_phase_index not in state.completed_steps:
            state.completed_steps[state.current_phase_index] = []
        state.completed_steps[state.current_phase_index].append(state.current_step_index)

        # 检查是否还有下一步
        if state.current_step_index + 1 < len(phase.steps):
            state.current_step_index += 1
            state.status = PipelineStatus.RUNNING
        else:
            # 当前 phase 完成
            state.completed_phases.append(state.current_phase_index)

            if phase.human_gate:
                # 需要人类审核
                state.status = PipelineStatus.WAITING_HUMAN
            elif state.current_phase_index + 1 < len(template.phases):
                # 进入下一 phase
                state.current_phase_index += 1
                state.current_step_index = 0
                state.status = PipelineStatus.RUNNING
            else:
                # 全部完成
                state.status = PipelineStatus.COMPLETED

        await self._save_state(state)

    async def handle_rejection(self, project_id: str, task_key: str):
        """处理审计封驳"""
        state = await self.get_state(project_id)
        if not state:
            return

        count = state.rejection_counts.get(task_key, 0) + 1
        state.rejection_counts[task_key] = count

        if count >= 3:
            # 连续 3 次封驳，标记需要人类介入
            state.status = PipelineStatus.WAITING_HUMAN
        else:
            # 重新执行当前步骤
            state.status = PipelineStatus.RUNNING

        await self._save_state(state)

    async def approve(self, project_id: str):
        """人类审核通过"""
        state = await self.get_state(project_id)
        if not state or state.status != PipelineStatus.WAITING_HUMAN:
            return

        template = get_template(state.template_name)
        if state.current_phase_index + 1 < len(template.phases):
            state.current_phase_index += 1
            state.current_step_index = 0
            state.status = PipelineStatus.RUNNING
        else:
            state.status = PipelineStatus.COMPLETED

        await self._save_state(state)

    async def _save_state(self, state: PipelineState):
        """保存状态"""
        await self.db.execute(
            """
            UPDATE pipeline_state SET
                status = $2, current_phase_index = $3, current_step_index = $4,
                completed_phases = $5, completed_steps = $6, rejection_counts = $7,
                updated_at = NOW()
            WHERE project_id = $1
            """,
            state.project_id, state.status.value,
            state.current_phase_index, state.current_step_index,
            json.dumps(state.completed_phases),
            json.dumps(state.completed_steps),
            json.dumps(state.rejection_counts),
        )


# 数据库表
PIPELINE_STATE_TABLE = """
CREATE TABLE pipeline_state (
    project_id UUID PRIMARY KEY REFERENCES projects(id) ON DELETE CASCADE,
    template_name VARCHAR(50) NOT NULL,
    status VARCHAR(20) DEFAULT 'not_started',
    current_phase_index INTEGER DEFAULT 0,
    current_step_index INTEGER DEFAULT 0,
    completed_phases JSONB DEFAULT '[]',
    completed_steps JSONB DEFAULT '{}',
    rejection_counts JSONB DEFAULT '{}',
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);
"""
```

#### 4.1.3 backend/core/pipeline/executor.py

```python
"""
Pipeline 执行器: 按模板驱动 Agent 执行。
"""

import asyncio
from .state_machine import PipelineStateMachine, PipelineStatus
from core.agent.registry import get_agent


class PipelineExecutor:
    """Pipeline 执行器"""

    def __init__(self, db, gateway, rag, storage):
        self.state_machine = PipelineStateMachine(db)
        self.gateway = gateway
        self.rag = rag
        self.storage = storage
        self.db = db

    async def advance(self, project_id: str) -> dict:
        """
        推进项目到下一步并执行。

        Returns:
            {"status": "completed"/"waiting_human"/"running", "result": ...}
        """
        state = await self.state_machine.get_state(project_id)
        if not state:
            return {"status": "error", "message": "Pipeline not initialized"}

        if state.status == PipelineStatus.WAITING_HUMAN:
            return {"status": "waiting_human", "message": "等待人类审核"}

        if state.status == PipelineStatus.COMPLETED:
            return {"status": "completed", "message": "流程已完成"}

        template = get_template(state.template_name)
        phase = template.phases[state.current_phase_index]
        step = phase.steps[state.current_step_index]

        # 执行当前步骤
        try:
            result = await self._execute_step(project_id, step, state)

            # 推进状态
            await self.state_machine.advance_step(project_id)

            new_state = await self.state_machine.get_state(project_id)
            return {
                "status": new_state.status.value,
                "phase": phase.name,
                "step": step.skill,
                "result": result,
            }

        except Exception as e:
            await self.state_machine.handle_rejection(
                project_id, f"{state.current_phase_index}-{state.current_step_index}"
            )
            return {"status": "error", "message": str(e)}

    async def _execute_step(self, project_id: str, step, state) -> dict:
        """执行单个步骤"""
        # 获取 Agent
        agent = get_agent(step.agent, self.gateway, self.rag, self.storage)

        # 构建输入
        task_payload = await self._build_payload(project_id, step, state)

        # 创建任务
        from core.agent.base import AgentTask
        task = AgentTask(
            task_id=f"{project_id}-{state.current_phase_index}-{state.current_step_index}",
            agent_name=step.agent,
            task_type=step.skill,
            project_id=project_id,
            payload=task_payload,
            cost_profile=step.cost_profile,
        )

        # 执行
        result = await agent.execute(task)

        # 写入结果
        await self._write_result(project_id, step, result)

        return result.data

    async def _build_payload(self, project_id: str, step, state) -> dict:
        """根据 step.input_from 构建 payload"""
        payload = {}

        for source in step.input_from:
            if source == "user_input":
                project = await self.db.fetchrow(
                    "SELECT * FROM projects WHERE id = $1", project_id
                )
                payload["user_requirements"] = dict(project) if project else {}
            elif source == "layer0":
                layer0 = await self.storage.get_layer0(project_id)
                payload.update({k: v.get("value", v) if isinstance(v, dict) else v 
                               for k, v in layer0.items()})
            elif source == "layer1":
                chars = await self.storage.get_character_states(project_id)
                payload["character_states"] = [dict(c) for c in chars]
                fs = await self.storage.get_foreshadow_states(project_id)
                payload["foreshadow_states"] = [dict(f) for f in fs]
            elif source == "layer3":
                outlines = await self.storage.get_chapter_outlines(project_id)
                payload["chapter_outlines"] = [dict(o) for o in outlines]
            elif source == "rag":
                pass  # RAG 由 Agent 内部的 _build_context 处理

        return payload

    async def _write_result(self, project_id: str, step, result):
        """将 Agent 结果写入对应的 Layer"""
        if step.output_to == "layer0":
            for key, value in result.data.items():
                await self.storage.update_layer0(project_id, key, str(value))
        elif step.output_to == "layer1":
            for key, value in result.data.items():
                if key.startswith("char_"):
                    await self.storage.update_character_state(key[5:], value)
                elif key.startswith("fs_"):
                    await self.storage.update_foreshadow_state(key[3:], value)
        elif step.output_to == "layer2":
            scene_id = result.data.get("scene_id")
            if scene_id:
                await self.storage.save_scene_draft(scene_id, result.data)
        elif step.output_to == "layer5":
            pass  # 索引由 RAG 的 post_process 处理
```

```python
"""
模板加载器: 从 YAML 文件加载编排模板。
"""

import yaml
from pathlib import Path
from .template import PipelineTemplate, Phase, Step


# 模板注册表
TEMPLATE_REGISTRY: dict[str, PipelineTemplate] = {}

TEMPLATES_DIR = Path(__file__).parent.parent.parent / "templates"


def load_templates():
    """加载所有 YAML 模板"""
    for yaml_file in TEMPLATES_DIR.glob("*.yaml"):
        with open(yaml_file, "r", encoding="utf-8") as f:
            data = yaml.safe_load(f)
        
        phases = []
        for phase_data in data.get("phases", []):
            steps = []
            for step_data in phase_data.get("steps", []):
                steps.append(Step(
                    agent=step_data["agent"],
                    skill=step_data["skill"],
                    input_from=step_data.get("input_from", []),
                    output_to=step_data.get("output_to", "next_step"),
                    condition=step_data.get("condition"),
                    cost_profile=step_data.get("cost_profile", "balanced"),
                    timeout=step_data.get("timeout", 600),
                ))
            phases.append(Phase(
                name=phase_data["name"],
                steps=steps,
                human_gate=phase_data.get("human_gate", False),
                max_parallel=phase_data.get("max_parallel", 1),
                repeat_until=phase_data.get("repeat_until"),
            ))
        
        template = PipelineTemplate(
            name=data["name"],
            description=data.get("description", ""),
            phases=phases,
            scale_config=data.get("scale_config", {}),
        )
        TEMPLATE_REGISTRY[template.name] = template


def get_template(name: str) -> PipelineTemplate:
    """获取模板"""
    if not TEMPLATE_REGISTRY:
        load_templates()
    template = TEMPLATE_REGISTRY.get(name)
    if not template:
        raise ValueError(f"Unknown template: {name}. Available: {list(TEMPLATE_REGISTRY.keys())}")
    return template


def list_templates() -> list[dict]:
    """列出所有模板"""
    if not TEMPLATE_REGISTRY:
        load_templates()
    return [
        {"name": t.name, "description": t.description, "phases": len(t.phases)}
        for t in TEMPLATE_REGISTRY.values()
    ]
```

## 五、成本优化策略

### 5.1 六大优化方案

#### 方案1: 审计降级

```
当前: 审计Agent全部用 DS-Reasoner
优化: 
  - 程序化检测器6项 → 代码执行，成本=0
  - 因果链推演/人设校验/对白风格 → DS-V4-Flash（缓存命中 ¥0.02/百万tokens）
  - 仅复杂推理场景升级到 DS-Reasoner
预估节省: 审计成本降 60-70%
```

#### 方案2: 创作分档

```
当前: 创作Agent一律用 DS-V4-Pro
优化:
  - 普通过渡场景 → DS-V4-Flash (Non-think)
  - 情感冲突场景 → DS-V4-Flash (Think High)
  - 哇塞时刻 → DS-V4-Pro (Think Max)
  - 对白生成 → MiMo-V2.5-Pro
预估节省: 创作成本降 40-50%
```

#### 方案3: 上下文缓存

```
当前: 每次调用都发 Layer 0 全文 (2-3万字)
优化: 
  - Layer 0 用 hash 校验，未变化时只发 fingerprint
  - 配合 DeepSeek Context Caching（缓存命中价仅 ¥0.02/百万tokens）
预估节省: 输入成本降 50-70%
```

#### 方案4: 程序化检测前置

```
当前: 审计Agent内先跑程序化检测再跑LLM
优化: 程序化检测在创作Agent提交后立即执行，失败直接打回不走LLM
预估节省: 减少 20-30% 无效LLM调用
```

#### 方案5: RAG 精准召回

```
当前: 固定 3万字上下文，按优先级裁剪
优化: RAG 检索 1-1.5万字精准上下文
预估节省: 输入 tokens 降 50%+
```

#### 方案6: 伏笔+创意合并调用

```
当前: 伏笔Agent和创意Agent分开调用各用 Think Max
优化: 合并为一次调用，输入高度重叠
预估节省: Phase 3 成本降 40%
```

### 5.2 成本对比（150万字剧本）

| 环节 | 当前方案 | 优化后 | 节省 |
|------|---------|--------|------|
| 创作Agent | ~$15-25 | ~$8-12 | 50% |
| 审计Agent | ~$8-15 | ~$3-5 | 65% |
| 伏笔+创意 | ~$3-5 | ~$2-3 | 40% |
| 状态+素材 | ~$1-2 | ~$1-2 | 0% |
| 上下文输入 | ~$5-8 | ~$1-3 | 65% |
| **合计** | **~$32-55** | **~$15-25** | **~50-55%** |

## 六、150万字极端场景适配

### 6.1 资源需求

| 资源 | 1万字 | 50万字 | 150万字 |
|------|-------|--------|---------|
| 场景数 | 10-50 | 150-200 | 300-400 |
| 伏笔数 | 5-20 | 20-60 | 60-178 |
| LLM调用次数 | 30-100 | 500-800 | 1200-2000 |
| 总tokens | 10-50万 | 500-1000万 | 2000-4000万 |
| Render Worker | 1个 | 2-3个 | 3-5个 |
| PostgreSQL存储 | <100MB | 1-3GB | 5-10GB |
| 月基础设施成本 | ~$0 | ~$50 | ~$125 |

### 6.2 关键瓶颈与解决方案

| 瓶颈 | 问题 | 解决方案 |
|------|------|---------|
| 上下文窗口 | 150万字 ≈ 300万tokens，超过1M窗口 | RAG 精准召回，不全量塞入 |
| Layer 1 膨胀 | 178条伏笔+400场景状态 | 只取出场角色和相关伏笔 |
| Celery超时 | 单场景流程可能超10分钟 | 分阶段执行，单步可重试 |
| Render部署套餐 | 资源与休眠策略取决于当前计划 | 以 `render.yaml` 和控制台实际套餐为准 |
| PostgreSQL | 免费层1GB限制 | 升级Standard 10GB |
| 前端渲染 | 400场景列表卡顿 | 虚拟列表+分页 |

### 6.3 分批并行策略

```
Worker 1: 生成 CH05-S03
Worker 2: 生成 CH05-S04 (不依赖S03全文，只依赖摘要)
Worker 3: 审计 CH05-S02 (上一轮)
Worker 4: 状态更新 CH05-S01 (审计通过)
```

## 七、RAG 集成方案

### 7.1 RAG 在系统中的定位

```
Layer 0-5 存储结构（不动）
        ↓
┌───────────────────────────┐
│     RAG 检索层（新增）       │
│  向量库: pgvector           │
│  索引内容:                  │
│  · 场景全文/摘要            │
│  · 角色设定                 │
│  · 伏笔定义                 │
│  · 世界观条目               │
│  · 信息点                   │
└───────────────────────────┘
        ↓
  素材Agent → 创作Agent / 审计Agent
```

### 7.2 索引时机

| 触发事件 | 索引内容 | 调用方式 |
|---------|---------|---------|
| 场景定稿 | 场景描述+对白+伏笔操作 | 自动（_post_process） |
| 角色更新 | 背景+动机+语言风格 | 自动 |
| 伏笔更新 | 三层定义+状态 | 自动 |
| 世界观锁定 | 各配置项 | 自动 |
| 项目创建 | 全量重建 | 手动触发 |

### 7.3 RAG 成本（150万字）

| 环节 | 次数 | 每次tokens | 单价 | 总成本 |
|------|------|-----------|------|-------|
| 索引 | 600次 | ~5000 | ¥1/百万 | ¥3 |
| 检索 | 800次 | ~3000 | ¥1/百万 | ¥2.4 |
| **合计** | | | | **¥5.4** |

RAG 节省的输入成本约 ¥100+，投入产出比 ≈ 1:20。

## 九、快速开始

### 9.1 本地开发

```bash
# 1. 启动 PostgreSQL + Redis
docker-compose up -d postgres redis

# 2. 初始化数据库（含 pgvector）
psql -f scripts/init_db.sql

# 3. 安装依赖
cd backend && pip install -r requirements.txt

# 4. 启动后端
uvicorn main:app --reload --port 8000

# 5. 启动 Celery Worker
celery -A tasks worker --loglevel info

# 6. 启动前端
cd frontend && npm install && npm run dev
```

### 9.2 创建第一个项目

```bash
# 创建互动影游项目（标准模式）
curl -X POST http://localhost:8000/api/projects \
  -H "Content-Type: application/json" \
  -d '{
    "name": "我的第一个影游",
    "template": "interactive_drama",
    "scale": "standard",
    "genre": "悬疑",
    "style": "暗黑",
    "target_word_count": 500000,
    "core_contradiction": "魔法的代价是记忆"
  }'

# 推进 Pipeline
curl -X POST http://localhost:8000/api/projects/{id}/pipeline/advance
```
